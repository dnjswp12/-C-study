- static 함수

static은 "정적"이라는 의미로, 프로그래밍 언어적으로는 함수가 끝나고 다시 불러졌을때

변수가 그 값을 유지하고 있도록 하기 위함이다. 

static 변수는 프로그램이 초기화 될때 메모리에서 제일 먼저 만들어둔다.

static이 붙은 함수는 함수가 선언된 파일 안에서만 함수를 호출 할 수 있다는 의미이다. 

아래와 같이 사용한다

- 
a.c

static int func();

int func()
{
  int a;
  ...
  
}

main.c

int main()
{
  func(); // 사용할 수 없음.
}
